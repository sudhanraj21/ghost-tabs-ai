# GhostTabs AI - Chrome Web Store Assets

## Screenshots Required

Chrome Web Store requires 2-5 screenshots (1280x800 or 640x400).

### How to Take Screenshots:

1. **Load the extension in Chrome:**
   - Go to `chrome://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked" → select the `dist` folder

2. **Settings page screenshot:**
   - Click the extension icon → "Settings"
   - Or right-click extension → "Options"
   - Take screenshot (Ctrl+Shift+S in Windows, Cmd+Shift+4 on Mac)

3. **Side panel screenshot:**
   - Click the extension icon in Chrome toolbar
   - The side panel shows your shelved tabs
   - Take screenshot

4. **Floating dock screenshot:**
   - Visit any website
   - Park a tab (click extension icon or use keyboard shortcut)
   - The floating dock appears at bottom of page
   - Take screenshot

### Save screenshots as:
- `store-assets/screenshot-settings.png`
- `store-assets/screenshot-sidepanel.png`  
- `store-assets/screenshot-dock.png`
